Sistema de Gestión Comercial (UX/UI + Desarrollo)

🇦🇷 Argentina

Keywords: DISEÑO UXUI / PROTOTIPOS / FIGMA / DIAGRAMA DE FLUJOS / USABILIDAD / ACCESIBILIDAD / HTML + CSS / JAVASCRIPT / FIREBASE / BASE DE DATOS

Diseñé en Figma y lo desarrollé en HTML, CSS, JavaScript y Firebase como base de datos. Se trata de una app web de gestión de ventas, control de stock, seguimiento de gastos y análisis económico del comercio. El objetivo era digitalizar y simplificar la información de venta del comercio. Con este sistema, se eliminara el uso del "cuadernito de ventas y gastos", mejorando así la experiencia de la Gerencia y del personal mediante la tecnología.
