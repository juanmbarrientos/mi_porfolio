LEGENDARYUM Wallet Cripto / NFTs (DISEÑADOR UX/UI)

Palabras claves: DISEÑO UXUI / PROTOTIPOS / FIGMA / DIAGRAMA DE FLUJOS / USABILIDAD / ACCESIBILIDAD / WALLET / CRIPTOMONEDAS

Además de diseñar la web principal y su entorno, recientemente me han encomendado el diseño de la UX y, posteriormente, la UI para una wallet cripto integrada en el sitio. Esto permitirá a los usuarios adquirir diversos assets y/o elementos dentro del mundo gaming y el metaverso, así también como enviar o recibir criptomonedas y NFTs.

Diseñé la wallet en dos versiones: una reducida solo con las funciones principales, y la otra en una versión ampliada que proporcionaba información con más detalle sobre las transacciones realizadas.

Fue fundamental tener en cuenta la seguridad y la forma de comunicar la interfaz al usuario para transmitir confianza en la realización de transacciones seguras.

PUNTOIA Wallet (DISEÑADOR UX/UI)

La empresa PuntoIA, con sedes en Colombia y Argentina, me contrató para liderar un importante proyecto: la creación de su propia Wallet. GRANJA LA PAULINA (UX/UI + Desarrollo)

Palabras claves: DISEÑO UXUI / PROTOTIPOS / FIGMA / DIAGRAMA DE FLUJOS / USABILIDAD / ACCESIBILIDAD / HTML + CSS / JAVASCRIPT / FIREBASE / BASE DE DATOS

Este es un proyecto freelance, diseñado en Figma y desarrollado con HTML, CSS, JavaScript y Firebase como base de datos. Se trata de una aplicación web para la gestión de ventas, control de stock, seguimiento de gastos mensuales y análisis económico del comercio. El objetivo es digitalizar y simplificar la información para acceder a los datos de manera más rápida y tomar decisiones informadas. Con este sistema, se eliminará el uso del "cuadernito de ventas y gastos", mejorando así la experiencia tanto de la Gerencia como del personal al simplificar procesos mediante la tecnología. Tanto la idea como el prototipo en Figma y el desarrollo de la programación han sido realizados de forma freelance.

TUCUMANTURISMO.COM.AR (UX/UI + Desarrollo)
