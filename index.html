LEGENDARYUM Wallet Cripto / NFTs (DISEÑADOR UX/UI)

Palabras claves: DISEÑO UXUI / PROTOTIPOS / FIGMA / DIAGRAMA DE FLUJOS / USABILIDAD / ACCESIBILIDAD / WALLET / CRIPTOMONEDAS

Además de diseñar la web principal y su entorno, recientemente me han encomendado el diseño de la experiencia de usuario (UX) y, posteriormente, la interfaz de usuario (UI) para una wallet cripto integrada en el sitio. Esto permitirá a los usuarios adquirir diversos assets y/o elementos dentro del mundo gaming y el metaverso, así también como enviar o recibir criptomonedas y NFTs.

Diseñé la wallet en dos versiones: una reducida solo con las funciones principales, y la otra en una versión ampliada que proporcionaba información con más detalle sobre las transacciones realizadas.

Fue fundamental tener en cuenta la seguridad y la forma de comunicar la interfaz al usuario para transmitir confianza en la realización de transacciones seguras.

Es un proyecto freelance, diseñado en figma y desarrollado html, CSS, Javascript y Firebase como base de datos. Es una app web de gestión de ventas, control de stock, ver los gastos del mes y la parte analitica y económica sobre el comercio.
